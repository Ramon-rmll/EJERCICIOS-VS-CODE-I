Elementos de bolque y linea

Una explicación de porqué algunos Elementos se comportan de manera distinta

El párrafo es un elemento de bloque y ocupa todo el ancho del navgegador; el siguiente elemento irá debajo

El div es otro elemento de bloque que ocupa todo el ancho del navegador

Los enlaces como las imágenes, son elementos de linea y van seguidos como las letras de una linea
